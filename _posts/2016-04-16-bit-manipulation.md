> Bit Manipulation

## 1. Operations for Bit
* & (And)
* | (OR)
* ^ (XOR)
* ~ (Negative)

## 2. Common Facts
x ^ 0s = x       x & 0s = 0      x | 0s = x
x ^ 1s = ~x      x & 1s = x      x | 1s = 1
x ^ x = 0        x & x = x       x | x = x
