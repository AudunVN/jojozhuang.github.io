This website is hosted by GitHub and powered by Jekyll. You can view this website through your desktop, laptop or other mobile devices, like pad or smart phones. Desktop and mobile web browsers with different size are all supported. All of the pages within this website are static. The blogs and portfolios are created with Markdown, check Markdown Cheatsheet.

Here are some links which helped me along the way:

  • Two blogs instruct how to setup website in Github, Creating and Hosting a Personal Site on GitHub and How I Created a Beautiful and Minimal Blog.
  • The official Jekyll website, along with detailed documentation.
  • Documentation by GitHub about hosting static webpages with GitHub pages.
  • The GitHub repository for my personal website.
  • The list page template for Portfolio.
  • The personal page template for CV.
